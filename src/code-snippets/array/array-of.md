## Create a new array, `[1, 2, 3, 4]`, using the `Array.of` method.

```javascript content
const arr = Array.of(1, 2, 3, 4);
```

## `console.log` the new array.

```javascript start
const arr = Array.of(1, 2, 3, 4);
```

```javascript content
console.log(arr);
```

```javascript after
// [1, 2, 3, 4]
```
